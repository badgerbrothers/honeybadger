# Feature: Artifact Management with MinIO Storage

## Feature Description

Implement artifact storage, download, and project saving functionality using MinIO (S3-compatible object storage). This enables task runs to generate and persist files (reports, screenshots, code, data) that users can download or save to their project directories.

## User Story

As a user
I want to store, download, and manage artifacts generated by task runs
So that I can access and organize the outputs produced by AI agents

## Problem Statement

Task runs generate various artifacts (reports, screenshots, code files, data), but currently there's no storage backend or API to:
- Upload artifacts to persistent storage
- Download artifacts
- List artifacts for a project or task run
- Save artifacts to project directories

## Solution Statement

Integrate MinIO as the object storage backend and implement REST API endpoints for artifact management. Use the minio Python client for S3-compatible operations. Artifacts will be stored with a structured path: `{project_id}/{task_run_id}/{artifact_id}/{filename}`.

## Feature Metadata

**Feature Type**: New Capability
**Estimated Complexity**: Medium
**Primary Systems Affected**: backend/app/routers, backend/app/services
**Dependencies**: minio (Python client, needs to be added)

### Patterns to Follow

**Router Pattern:**
```python
# From tasks.py:10-20
router = APIRouter(prefix="/api/artifacts", tags=["artifacts"])

@router.get("/{artifact_id}", response_model=ArtifactResponse)
async def get_artifact(artifact_id: uuid.UUID, db: AsyncSession = Depends(get_db)):
    result = await db.execute(select(Artifact).where(Artifact.id == artifact_id))
    artifact = result.scalar_one_or_none()
    if not artifact:
        raise HTTPException(status_code=404, detail="Artifact not found")
    return artifact
```

**Configuration Pattern:**
```python
# From config.py
class Settings(BaseSettings):

    minio_access_key: str = "minioadmin"
    minio_secret_key: str = "minioadmin"
    minio_bucket: str = "artifacts"
    minio_secure: bool = False
```
